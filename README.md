# ColorQuest 🎨

A color memory game that tests your visual perception.

You're shown a color for 10 seconds ; then it disappears. Using a color picker, recreate the color from memory. After 5 rounds, see how well your eyes really work.

## How to Play

1. Click **Start Game**
2. Memorise the color shown for 10 seconds
3. Use the color picker to match it from memory
4. Hit **Verify** to see your score
5. Complete all 5 rounds to get your final result

## Scoring

- **80%+** : Colour Master
- **60–79%** : Sharp Eye
- **Below 60%** : Keep Practising

## Tech Stack

- React + Vite
- Vanilla CSS (no UI libraries)
- HSV color model for accurate color picking
- Perceptually weighted color similarity scoring

## Run Locally

```bash
git clone https://github.com/Ana-m7/ColorQuest.git
cd ColorQuest
npm install
npm run dev
```
